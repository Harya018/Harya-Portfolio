Jeeva Raksha: AI Animal Rescue Dispatch

AI-powered rescue dispatch platform connecting citizens reporting injured animals with nearby rescuers via chatbot (text, voice, and image input).

Computer Vision GPS Tracking Python LLM Flask
🐙 GitHub
Architecture

Multi-modal chatbot frontend (React/HTML) → Flask REST API → LLM router → Computer Vision model (YOLOv8) for animal detection → GPS-based rescuer matching → Notification pipeline.

Key Challenges

Real-time geolocation matching with low latency; multi-modal input fusion (text + image + voice); ensuring LLM responses stay contextually grounded to rescue scenarios.
